◆ GetSystemConfigPath()

std::string EnvironmentSetup::GetSystemConfigPath ( )
static

获取跨平台的系统级配置文件目录路径。

  • 在 Windows 平台上，此函数返回 APPDATA% 环境变量指向的路径。
  • 在 Linux 和 macOS 平台上，此函数遵循 XDG 基本目录规范，返回 $HOME/.config 路径。 该函数的性能非常高，因为它主要依赖于读取环境变量，这是一个低开销的操作。
返回
返回一个 std::string，其中包含配置目录的绝对路径。 如果无法确定路径（例如，所需的环境变量未设置），则返回一个空字符串。
