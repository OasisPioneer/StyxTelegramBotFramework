Styx Telegram Bot Framework

A modular Telegram bot framework for C++ developers.

Describe

‍This project is a Telegram bot framework designed for C++ developers, aimed at simplifying the complexities of bot development. By leveraging a highly modular plugin system, you no longer need to worry about low-level network communication or API interactions. Simply focus on developing plugins for your desired features, and you can quickly build powerful and extensible Telegram bots.
