## 描述

> 本项目是一个为 C++ 开发者设计的电报 (Telegram) 机器人框架，旨在解决原生 API 开发的复杂性。通过高度模块化的插件系统，您无需再关心底层的网络通信和
> API 交互，只需专注于实现具体功能的插件，即可快速构建功能强大、易于扩展的电报机器人。
